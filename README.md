# 🟤 Material You NewTab  — A Minimal Aesthetic Browser Experience

> Transform your default new tab into a calm, clear, distraction-free workspace.

---

## ✨ Overview

**Material You NewTab is a modern browser new tab interface inspired by the clean aesthetics of Chrome themes.**
It replaces the default cluttered tab with a **minimal, visually soothing environment** designed for focus and simplicity.

Built with performance and design in mind, this project delivers a smooth and elegant browsing experience without unnecessary complexity.

---

## 🚀 Features

* 🎨 Material You NewTab** Theme UI** — Warm, eye-friendly tones for long usage
* ⚡ **Fast & Lightweight** — No unnecessary scripts or heavy assets
* 🧩 **Browser Extension Ready** — Works as a Chrome/Edge new tab replacement
* 🧘 **Distraction-Free Design** — Focus on what matters
* 💡 **Modern UI/UX** — Clean layout with subtle visual polish
* 📱 **Responsive Layout** — Works across screen sizes

## 📦 Installation (Extension Setup)

1. Download or clone this repository
2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder
6. Done — your new tab is now replaced 🎉

## 🎯 Purpose

Most new tab pages are either:

* Overloaded with widgets
* Or completely useless

**Material You NewTab aims to strike the balance:**

> Minimal enough to stay clean, functional enough to be useful.
